# 极简图片浏览器

## 需求

- Windows原生应用
- 极小、极快、秒开
- 可浏览JPG、PNG等常见格式的图片
- 方向键（上下、左右）浏览上一张、下一张
- Page Up放大，Page Down缩小
- 软件打开后，双击空白区选择图片
- 默认自适应窗口大小显示图片，鼠标拉动改变窗口大小时，图片自适应缩放
- 如果图片包含webui兼容的图片生成信息，则在界面右侧显示表格形式的面板，显示图片生成信息。如果不包含，则不显示该面板。
- 信息面板仅显示webui兼容信息，不需要显示comfyui的workflow信息。
- 信息面板必须是表格样式，2列：第一列是key，第二列是值。值文字较多时，自动换行显示。
- 点击信息面板的表头可以将表格缩起，再次点击可以重新打开表格信息面板。
